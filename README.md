# Class Schedule Builder Mobile App

A student-centric Android application designed to streamline academic scheduling and prevent time conflicts.

## 🚀 Key Features
*   **Conflict Detection:** Prevents overlapping class times with intelligent real-time scheduling checks.
*   **Offline Access:** Utilizes Room Database for persistent local data storage without requiring an internet connection.
*   **Secure Authentication:** Integrated Firebase Authentication for secure user login and management.
*   **Responsive UI:** Designed an intuitive, mobile-optimized timetable grid using Material Design principles.

## 🛠️ Tech Stack
*   **Platform:** Android (Java)
*   **Database:** Room DB (SQLite)
*   **Authentication:** Firebase
